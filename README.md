# GainPath — AI Workout Tracker

A free, open-source workout tracking web app for Push/Pull/Legs/Upper/Lower training splits.

## Features
- 🏋️ Male and female workout plans (PPL + Upper/Lower)
- 🤖 AI coaching feedback after every set — powered by Claude
- 💪 Per-exercise RPE rating with instant next-session recommendations
- ⏱️ Rest timer with adjustable duration
- 📈 Progress charts — track weight lifted over time
- 🏆 Personal record (PR) tracker with automatic detection
- 🔥 Streak counter
- 📄 Monthly PDF report — calendar, strength gains, new PRs
- 💾 Backup and restore your data as JSON
- 📱 Works on mobile — add to home screen from Safari or Chrome

## How to use
1. Open the app at [jedmangubat.github.io/gainpath](https://jedmangubat.github.io/gainpath)
2. Complete the one-time setup (name, sex, goals, preferences)
3. Pick your training day and start logging sets
4. Rate how each exercise felt — AI adjusts your next weights automatically

## Tech
Built as a single HTML file. No installation, no backend, no account needed. Data saved locally in your browser.
